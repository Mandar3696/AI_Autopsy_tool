AI Autopsy Tool is a web-based application that analyzes images using AI
to detect manipulated, fake, or AI-generated content.

## Features
- Image upload and analysis
- AI-based detection
- Authentication (Login / Logout)
- Secure backend

## Tech Stack
- Frontend: React / Vite
- Backend: Python
- AI/ML: TensorFlow / OpenCV
- Auth & DB: Supabase

## How to Run
1. Clone the repo
2. Install frontend & backend dependencies
3. Add `.env` file
4. Run frontend and backend servers
